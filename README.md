# Honkai-StarRail-Daily-Quests  
基于pyautogui的自动化每日刷取体力脚本  

第一次使用请确保您电脑上拥有Python  
然后运行Install_Requirements.py  
-----------------------------------
以后只需要运行Launch.py即可  
开发日志  
v1.0  
功能：刷取拟造花萼（金）以及侵蚀隧洞的死水，大公和信使本  
P.S.目前仅支持中文客户端
